# Contributing

## Reproduce the analysis

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter notebook heart_disease_imbalanced_classification.ipynb
```

Run the notebook top to bottom. Figures under `figures/` are regenerated by the plotting cells.

## Data

`data/heart_2020_cleaned_edited.csv` is a 3,087-row subset of the BRFSS-derived heart disease survey data used for the write-up. Keep preprocessing changes documented in the notebook so metrics stay comparable.
